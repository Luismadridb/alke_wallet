# Proyecto Wallet – Módulo 2  

**Fundamentos del Desarrollo Frontend**

Este repositorio contiene el desarrollo del proyecto Wallet, realizado como parte del Módulo 2 , cuyo objetivo es aplicar los fundamentos de HTML, CSS, JavaScript básico, Bootstrap y jQuery.

---

## Objetivo del Proyecto

Construir una aplicación web simple que simule el funcionamiento básico de una wallet digital, permitiendo:

- Iniciar sesión
- Navegar entre distintas pantallas
- Simular operaciones como:
  - Depósito
  - Envío de dinero
  - Visualización de saldo
- Aplicar estilos responsivos
- Incorporar interactividad básica con JavaScript y jQuery

---

## Tecnologías Utilizadas

- **HTML5**  
- **CSS3**
- **JavaScript (básico)**
- **Bootstrap 5**
- **jQuery**

## Navegación

Todas las pantallas incluyen un navbar responsive, desarrollado con Bootstrap, que permite:

- Navegar entre las distintas vistas
- Adaptarse a pantallas móviles mediante menú hamburguesa

---

## Diseño y Estilos

- Se utilizó Bootstrap para:
  - Grillas
  - Formularios
  - Botones
  - Navbar responsive
- Se agregó un archivo `styles.css` para personalizar:
  - Contenedores principales
  - Tarjetas (`card`)
  - Espaciados y alineación
- El diseño busca ser claro, limpio y funcional.

---

## Lógica JavaScript

La lógica implementada cumple estrictamente con los contenidos del módulo:

- Captura de eventos (`click`, `submit`)
- Uso de:
  - `document.getElementById`
  - `addEventListener`
  - `alert()`
  - `window.location.href`

---

## Cómo Ejecutar el Proyecto

1. Clonar el repositorio
2. Abrir el archivo `login.html` desde la carpeta `assets/html`
3. Navegar entre las distintas pantallas usando el menú
